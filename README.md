🧩 项目概述:

这是一个用于 验证 AI 抽取的文档条款结构一致性 的 Web 工具。用户可以输入或上传包含条款信息的 JSON 数据，系统会自动分析并展示条款结构的完整性和潜在问题

💡 功能介绍

JSON 文档输入
支持用户粘贴 JSON 内容或上传 JSON 文件进行分析。

条款结构验证
工具会解析条款数据并检查每条内容的结构是否符合预期逻辑（例如是否缺失字段、层级是否正确等）。

结果展示区域
分析后会在界面中展示结构验证结果，方便用户快速定位问题。

多语言支持切换
提供中英文切换按钮（EN / 中文）方便不同语言用户使用。

🧠 预期用途

用于 AI 信息抽取后对条款结构的自动 QA（质量检查）

支持法务、产品、数据团队检视大型文档数据的结构一致性

可作为条款、合同数据预处理与清洗的前置步骤

🚀 使用说明
1. 打开工具

访问 Web 界面（例如：https://qa-webtool--harborbai02.replit.app/app）。

2. 输入或上传 JSON

在文本框中粘贴 JSON 格式条款数据

或者使用上传功能选择本地 JSON 文件进行分析

3. 运行结构验证

点击 “Analyze/验证” 按钮（或类似操作按钮）开始分析过程。

4. 查看结果

结果区域会展示解析的条款结构和检测到的问题（如缺失字段、格式错误等）。

📦 技术栈（示例）

以下示例为常见 Web 应用组合，具体以实际实现为准：

前端：HTML / CSS / JavaScript（可能使用 React 或其他框架）

后端：Node.js 或 Python Web 服务

部署平台：Replit（实时部署并可直接访问网址）

🏷 项目优势

操作简单，无需编程背景

可提升 AI 抽取结果的信任度

快速验证大量条款 JSON 数据

📌 注意事项

当前版本主要针对结构一致性检查，不会自动修复错误

JSON 数据格式需符合预期的约定（例如必须包含条款标题、内容字段等）
